My Projects

Computational Fluid Dynamics (CFD)

This project utilizes advanced Computational Fluid Dynamics (CFD) to simulate a two-dimensional mixing chamber driven by rotating disks, solving the Navier-Stokes, continuity, and mass fraction transport equations to capture complex fluid flow and scalar distribution.

Download Project Report (PDF)

NASA Propellant Feed System

This project provides a comprehensive analysis of the RS-25 rocket engine's propellant feed system, focusing on the flow paths of liquid hydrogen and liquid oxygen, their combustion in the preburner, and injection into the main combustion chamber at full power. It systematically examines each component—including fluid properties, turbo pumps, flow splits, preburner dynamics, main injectors, thrust chamber, nozzle design, and thrust calculations.

Download Project 2 Report (PDF)

VAD Lidar

This project analyzes wind data from a specific site using Vector Azimuth Display (VAD) to extract wind speed and direction profiles over time and height.

Download Project 3 Report (PDF)

Machine Training Model Images for Cat and Dogs

It's important to distinguish between species; we all know what a cat and dog looks like, but a machine doesn’t. In this project, I program in Python an application that is able to tell the difference between a cat and dog using convolutional and pooling layers. The model was trained with 25,000 pictures of cats and dogs.

Geothermal U-tube Thermal Performance

This project models a vertical geothermal U-tube heat exchanger using OpenFOAM. The simulation evaluates heat transfer performance over time by coupling fluid flow in the tube with conduction in the surrounding ground, informing efficiency for sustainable heating and cooling applications.
