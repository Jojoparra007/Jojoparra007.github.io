Welcome

Welcome to the Soft Robotics Lab Series, a hands-on guide to designing and simulating a soft robotic gripper using ANSYS Mechanical and MATLAB (SoRoSim). Here, you'll find step-by-step instructions, downloadable resources, and practical tips for anyone looking to explore the world of soft robotics.

Project Overview

The primary objective of this series is to model a pneumatic soft robotic gripper with four fingers, capable of picking up a round cross-section object. Across four labs, you will learn how to:

  • Set up the geometry, mesh, and material properties in ANSYS
  • Apply boundary conditions, loads, and contact interactions
  • Assemble multiple soft actuator segments into one gripper
  • Optimize designs by analyzing simulation output and refining parameters

By the end of these labs, you will have a clear understanding of how to use Finite Element Analysis (FEA) tools and techniques to model and optimize soft actuators in a professional engineering environment.

Lab 1 Prenumatic Hand Gripper

Lab 1: Intro to Soft Robotics & SoRoSim (MATLAB)

Goal: Provide an introduction to continuum soft robots and demonstrate how to set up a basic soft actuator simulation using SoRoSim in MATLAB.

  1. Background: Quick overview of continuum robotics and the benefits of pneumatic soft actuators.
  2. SoRoSim Setup: Installation and dependencies, plus a sample script to run a simple bending actuator model.
  3. Key Parameters: How pressure, segment length, and stiffness affect the robot’s bending behavior.
  4. Visualization: Generating plots or animations showing actuator deformation.

Tip: If you’re new to MATLAB, focus on the basics: how to run .m files, navigate the workspace, and interpret command window outputs.

Sorosim

Lab 2: ANSYS Setup – Geometry & Meshing

Goal: Teach how to model a soft robotic finger in ANSYS, focusing on creating and meshing the geometry.

  1. Geometry Creation: Sketch or import the 3D model of a soft finger with an internal channel for pressurization.
  2. Meshing Strategy: Choose suitable element types (e.g., hexahedral/tetrahedral) and refine the mesh for accurate bending simulations.
  3. Quality Checks: Check aspect ratio, skewness, and element count to ensure stable solutions.

Tip: Keep the mesh coarse for quick test runs, then refine it for final simulations.

Lab 3: Material Modeling & Assembly

Goal: Introduce hyperelastic materials in ANSYS and demonstrate how to assemble multiple soft fingers into one gripper.

  1. Material Models: Use hyperelastic material data (e.g., Neo-Hookean, Ogden, or Mooney-Rivlin) to simulate silicone-like behavior.
  2. Assembly: Combine four finger models into a single gripper, ensuring proper alignment and contact between parts.
  3. Connections & Constraints: Define how each finger attaches to a base or manifold for uniform pressure distribution.

Tip: If you have real-world material test data, use the curve-fitting tools in ANSYS to get more accurate results.

Lab 4: Simulation & Post-Processing

Goal: Apply internal pressure loads to the assembled gripper, run the simulation, and optimize the design based on the results.

  1. Pressure Loading: Assign internal pressure in each finger’s chamber.
  2. Nonlinear Simulation: Solve for large deformations and contact with a round object.
  3. Results Analysis: Evaluate stress, strain, displacement, and contact forces. Compare predicted gripping force vs. real requirements.
  4. Design Optimization: Adjust geometry, material thickness, or pressure values to achieve the best grip performance.

Tip: Experiment with different pressures and note how the finger curvature changes. This helps you understand real-world design trade-offs (e.g., higher pressure = stronger grip but also more stress).
